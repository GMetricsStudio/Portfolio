FITNESS APP - Complete Authentication System

Run Instructions:
1. Desktop (Windows/Mac/Linux):
   - Run: ./run_desktop.sh  (or python FrameWork.py)
   - Requires: Python 3.9+, PySide6

Features:
- Complete user registration with email/password
- Secure local storage (data/accounts.json)
- Login system with validation
- User profile page
- Onboarding screens
- Responsive iPhone-like UI (375x812)

Data Storage:
- All user data stored locally in data/accounts.json
- Passwords are hashed (SHA-256)
- No external servers required
